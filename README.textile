h2. bin/setup

bin/setup copies files, listed in dotfiles/, FROM YOUR $HOME, overriding those in dotfiles/. If file is not found in $HOME.

Then it softlinks files from dotfiles/ to your $HOME thus you're ready using git to control your domestic dotfiles.
